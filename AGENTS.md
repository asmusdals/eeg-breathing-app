# UI design rules for this project

Goal:
Make the app feel premium, clean, aesthetic, modern, and calm.

Visual direction:
- soft minimal dashboard aesthetic
- cool light gray-blue background
- off-white elevated cards
- subtle shadows only
- minimal borders
- restrained blue accent
- elegant rounded corners
- strong but calm typography hierarchy
- spacious layout with lots of whitespace
- consistent spacing and alignment
- cohesive, high-end SaaS/productivity look

Avoid:
- orange gradients
- glossy surfaces
- glassmorphism
- too many nested cards
- harsh blacks
- too many accent colors
- oversized controls
- noisy UI
- inconsistent spacing

Implementation rules:
- preserve functionality
- preserve responsiveness
- prefer reusable components over one-off styles
- reduce visual complexity
- keep code organized and readable
- extract presentation components when it improves consistency
